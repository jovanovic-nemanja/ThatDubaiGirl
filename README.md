Laravel backend and API for frontend

1,   .env
2,   composer install, composer require stripe/stripe-php
3,   php artisan migrate --seed
4,   npm install laravel-mix --save-dev
	npm install vuex --save
5,   npm cache clear --force
6,   npm install
7,   npm run dev(watch)
8,   php artisan serve
